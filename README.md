## 설치 및 실행 방법
1. Repository Clone
```
git clone https://github.com/namkikim0718/todo-react.git
```

2. 의존성 설치
```
npm install
```

3. 실행하기
```
npm start
```
4. 브라우저에서 열기
```
http://localhost:3000
```
